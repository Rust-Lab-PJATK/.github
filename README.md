# Rust Lab PJATK

**Rust Lab PJATK** to studenckie koło naukowe Polsko-Japońskiej Akademii Technik Komputerowych, skupione na popularyzacji języka programowania Rust oraz tworzeniu w nim wydajnego i bezpiecznego oprogramowania [rust-lab-pjatk.github.io](https://rust-lab-pjatk.github.io/). Najważniejsze projekty organizacji to:

-   [sirtck_exercises](https://github.com/Rust-Lab-PJATK/sirtck_exercises) – zbiór zadań programistycznych w języku Rust, wykorzystywany wraz z narzędziem **SIRTCk** (*Simple Interactive Rust Task Checker*) do automatycznego sprawdzania rozwiązań.
    
-   [hachi](https://github.com/Rust-Lab-PJATK/hachi) – nowoczesny, wieloplatformowy emulator CHIP-8 napisany w Ruście, zaprojektowany do wiernej emulacji klasycznej maszyny wirtualnej z lat 70. XX wieku.
    
-   [geirmund](https://github.com/Rust-Lab-PJATK/geirmund) – projekt rozwijający architekturę rozproszoną z własnym protokołem komunikacji (model *master–worker*) w języku Rust, eksplorujący zagadnienia współbieżności i systemów rozproszonych.
    
-   [exercises](https://github.com/Rust-Lab-PJATK/exercises) – otwarte repozytorium ćwiczeń programistycznych do nauki języka Rust, służące doskonaleniu praktycznych umiejętności w tym języku.
